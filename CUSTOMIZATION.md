# CUSTOMIZATION GUIDE

This document explains how to personalize and maintain your GitHub profile over time.

## Profile Banner
Replace:
assets/banner.svg

You can update:
- Name
- Title
- Tagline
- Accent color

Recommended style:
- Background: #0D1117
- Accent: #58A6FF
- Font: Inter or Segoe UI

---

## README

Update these sections regularly:

### Current Focus
Change this whenever you begin a new project.

### Featured Projects
Keep only your best 2–4 projects.

Recommended order:
1. DEADRYX
2. AI Finance Manager
3. Portfolio
4. DSA Python

### Tech Stack

Only include technologies you've actually used in projects.

Avoid adding tools just because you've watched a tutorial.

### Certifications

Replace the placeholder with:
- Certificate name
- Issuer
- Date
- Verification link (if available)

### GitHub Stats

These update automatically.
No changes required.

---

## Repository Standards

Every repository should contain:

README.md
LICENSE
.gitignore
Screenshots
Installation Guide
Tech Stack
Future Roadmap

---

## Commit Messages

Examples:

feat: add workout tracking

fix: resolve login issue

docs: update README

refactor: improve project structure

style: improve UI spacing

## Profile Maintenance

Weekly:
